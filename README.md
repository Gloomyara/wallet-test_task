# wallet-test_task

## Стек Технологий
- Java 17
- Maven 3.9.6
- Spring Boot 3.2.4
- Spring Data
- PostgreSQl
- MapStruct
- Lombok
- Docker

## Сборка
1. Клонируйте репозиторий:
```Bash
git clone https://github.com/Gloomyara/wallet-test_task.git
```
2. Перейдите в каталог проекта: 
```Bash
cd wallet-test_task
```
3. Скомпилируйте исходные файлы:
```Bash
mvn clean package
```
4. Запустите проект:
```Bash
docker-compose up
```
